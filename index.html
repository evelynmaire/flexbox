Flexbox - the fuck you up sQUAD edition

Flexboxes are super handy when it comes to making responsive websites, so I made myself a reference page for all the different ways you can apply and sort them. I'm also figure skating trash, so.

Let's Get Started!

HTML

<div class="gallery">
  <div><img src="images/boyang.jpg" alt="Boyangsta"></div>
  <div><img src="images/shoma.jpg" alt="Shomajesty"></div>
  <div><img src="images/nathan.jpg" alt="Slaythan"></div>
</div>

CSS

.gallery {
  display: flex;
  flex-wrap: wrap;  /* STACKS UNDERNEATH WHEN SCREEN RESIZED FOR MOBILES */
  background: cornflowerblue;   /* REMOVE THIS IF YOU DON'T WANT A BACKGROUND COLOR */
}

.gallery img  {
  display: block;   /* GETS RID OF SMALL SPACE AT BOTTOM OF LOWEST IMAGE */
}

Flex Alignment

Flex End

.gallery {
  justify-content: flex-end;
}

Flex Center

.gallery {
  justify-content: center;
}

Flex Space-Between

.gallery {
  justify-content: space-between;
}

Flex Space-Around

.gallery {
  justify-content: space-around;
}

Flex Ordering

Specific Order

<div class="gallery">
  <div><img src="images/boyang.jpg" alt="Boyangsta"></div>
  <div class="eat-your-veggies"><img src="images/shoma.jpg" alt="Shomajesty"></div>
  <div><img src="images/nathan.jpg" alt="Slaythan"></div>
</div>
.gallery .eat-your-veggies {
  order: -1;    /* THIS CHANGES THE ORDER OF THE IMAGES, USEFUL FOR CHANGING ORDER FOR REPSPONSIVE DESIGN */
}

Flex Direction

Reverse order

.gallery {
  flex-direction: row-reverse;
}

Column

.gallery {
  flex-direction: column;
}

Column Center Align

.gallery {
  flex-direction: column;
  align-items: center;
}

Flex Alignment

Flex End Alignment

.gallery {
  align-items: flex-end;
}

Flex Center Alignment

.gallery {
  align-items: center;
}
